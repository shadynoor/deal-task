Form Content

  • Avatar Photo with Camera or upload from Gallery
  • Input text for user name with minlength 3 and maxlength 10
  • Voice Recorder for the user to record an introduction about himslef
  • Audio Player for recorded audio
  • Submit Button to check form validity and submit

Notes

  1. After Build For Android you have to add android record permission in AndroidManifst.xml, it will be found in assets.
  2. For IOS Build you have to add NSMicrophoneUsageDescription In info.plist
  3. This already tested on Web and Android, but not IOS cuz I don't have Mac OS
  4. Data Page is just to preview all stored data
  5. Form Page restore last saved data and it will fetch it once the connection is restored
  6. To update user photo click on the image it self and it shall open action sheet to select camera or upload from gallery
  7. Images converted to base64 to be saved in local storage and to preview
  8. Name input and image are required
  9. Sound Recorder is required and have a preview after record
  10. Record converted to base64 to be saved in local storage and to preview
  11. After stop the record it stops mic stream
  12. Any error appear as a toast message
  13. All data will be saved in local storage if the whole data is valid
  14. lazy loading technique implemented
  15. SSR implemented
